## 1. CTR 후킹 — 스크롤 멈추게 하는 요소

**🔥 핵심 발견: "제품 덜 보여주기"**

저희 데이터에서 CTR과 가장 강한 상관관계:
| 변수 | 상관계수 | 해석 |
|------|---------|------|
| **제품발견성** | **-0.535** | 제품 안 보일수록 CTR↑ |
| 긴급성 | +0.221 | 긴급할수록 CTR↑ |
| 심플함 | +0.128 | 심플할수록 CTR↑ |

**제품발견성별 CTR:**
| 제품발견성 | CTR | ROAS |
|-----------|-----|------|
| **1-3점 (안보임)** | **3.16%** | **2.42** |
| 7-10점 (잘보임) | 0.92% | 1.75 |

→ **제품 덜 보여주고, 사람/상황에 집중!**

에르니에 적용하면: 세제 병 크게 찍지 말고, **아기 빨래하는 엄마 손** 같은 장면 위주로.

## 2. 안전/신뢰 소구 — 기능 강조 없이 CTR 올리기

저희도 임산부 대상이라 "안전" 중요해요. 효과 있었던 방식:

**A. 인스타후기 타입 (CTR 1.45%)**
- SNS 캡쳐 느낌
- "이미 쓰고 있지" 같은 대세감
- 실제 사용자 느낌

**B. 스토리텔링 (CTR 2.6%)**
- "임산부가 발명한" 대표님 스토리
- 제품 기능 말고 사람 이야기

**C. 비교컷 (CTR 1.28%)**
- 전후 비교는 "기능"이 아니라 "변화"를 보여줌

에르니에 적용: "EWG 1등급" 말고 **"아기 피부 걱정 끝났어요"** 같은 결과 중심.

## 3. 명도대비 — 안전한 느낌 vs 주목도

**저희 결론: ROAS에는 7-8, CTR에는 큰 차이 없음**

| 명도대비 | CTR | ROAS |
|----------|-----|------|
| ~6 | - | 1.54 |
| **7-8** | - | **2.00** |
| 9-10 | - | 1.78 (싸구려 느낌) |

명도대비-CTR 상관: -0.092 (거의 없음)

**제안:**
- "안전한 느낌"이 브랜드라면 5-6으로 가도 CTR 손해 없음
- 대신 **첫인상임팩트**를 다른 요소로 보완 (카피 크기, 모델 표정)

## 4. 이미지 타입별 CTR

저희 데이터:
| 이미지타입 | CTR | ROAS |
|-----------|-----|------|
| **인스타후기** | **1.45%** | 1.37 |
| 비교컷 | 1.28% | 1.55 |
| 모델컷 | 1.09% | 1.69 |
| 연출컷 | 1.06% | **2.25** |
| 제품컷 | 0.75% | 1.76 |

**CTR은 인스타후기 > 비교컷 > 모델컷**
**ROAS는 연출컷이 최고**

→ 인스타후기로 클릭 유도 + 랜딩에서 전환 잡는 구조가 맞을 듯.

## 5. 텍스트 양 — 많이 vs 적게

직접 분석은 없지만, **심플함 +0.128** 상관에서 힌트.

저희 CTR TOP3 소재 카피비율:
- 1위 (CTR 6.5%): 20%
- 2위 (CTR 3.6%): 30%
- 3위 (CTR 3.0%): 20%

→ **20-30%가 최적**. 텍스트 너무 많으면 산만.

## 💡 에르니 전용 제안

저희 데이터 + 에르니 상황 종합:

1. **제품 안 보여주기**: 세제 병보다 **아기 빨래/엄마 손/깨끗한 옷**
2. **인스타후기 타입 테스트**: "이 세제 쓰고 피부 걱정 끝" 같은 SNS 캡쳐

4. **카피 20-30%**: 심플하게
5. **할인 병행**: 저희도 긴급성 +0.221 효과 있었음. "자연 감성 + 할인"이 CTR 3.3% 나왔다고 하셨잖아요.
