---

### 제외 기준

랜딩 URL에 `category_no=120` 포함된 소재 전부 제외

### 왜?

- 특가전/기획전 소재는 일반 소재와 패턴이 다름
- 이거 섞어서 분석하면 허상에 fit됨
- 바필이 오늘 이거 때문에 삽질함 (CTR 분석 틀림)

### 해야 할 것

1. **DB 점검** — category_no=120 들어간 거 전부 찾기
2. **제외 컬럼 추가** — "특가전제외" 같은 플래그
3. **분석할 때 항상 제외** — 필터 걸고 시작

### 체크 방법

```python
# 랜딩 URL에서 체크
df["특가전"] = df["landing_url"].str.contains("category_no=120", na=False)
df_clean = df[df["특가전"] == False]
```
