## 새 구조

```
Gemini Gem = 기획 (카피 + GPT 프롬프트 설계)
    ↓
GPT 1.5 = 실행 (이미지 생성)
    ↓
캠마 = 관리 (데이터 분석, ROAS 모델 업데이트)
```

---

## 역할 분담

### Gemini Gem (기획)
- 카피 작성 (원칙 기반)
- GPT 1.5용 프롬프트 생성
- 이미지 태깅 (71개)
- ROAS 예측 (v14)

**장점:** 지식파일 저장됨 → 매번 주입 안 해도 됨

### GPT 1.5 (실행 = 일러스트레이터)
- 실사 사진 + 디자인 요소 합성
- 프레임/배경/타이포 배치
- 배너 이미지 생성

**장점:** 이미지 생성 퀄리티 좋음, 일러스트레이팅 강점

### 캠마 (관리 = 백엔드)
- 데이터 분석
- ROAS 모델 업데이트
- 가설 검증
- Gem 유지보수
