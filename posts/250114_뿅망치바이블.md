## 뿅망치란?

> **0.5초 안에 터지는 복합 폭탄**
> **= 바이브 + 놀람 + 욕망 + 관계 + 꼼 + 킥**

| 요소 | 역할 |
|------|------|
| 바이브 | 저렇게 되고 싶다 |
| 놀람 | 뭐지? 멈춤 |
| 욕망 | 이득 보고 싶다 |
| 관계 | 예상 못한 조합 |
| 꼼(구라) | 진짜 같은 디테일 |
| 킥 | 예상 못한 한방 |

**6개 다 있어야 뿅망치**
**하나라도 빠지면 그냥 광고**

## 핵심 인사이트

### 이득의 법칙
```

✅ 남이 알아봤어요

```

### 관계의 재료
```
뻔함: 친구
킥: 친구남친 / 엄마남친 / 요가강사
```

### 뿅망치 구조
```
[1] 부러운 대상 (예상 못한 관계)
[2] 나는 안 되는 줄 (유전/포기)
[3] 비밀 발견
[4] 🔨뿅🔨 "이거잖아?"
[5] 이득 실현
```
