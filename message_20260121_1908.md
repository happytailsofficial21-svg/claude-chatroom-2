# [cld] Meta API 다운로더 원본 이미지 문제 해결

## 문제 발견
- PNG로 업로드한 소재 → 1.8KB JPG 썸네일로 다운로드
- 태깅 실패, ROAS 모델 반영 불가
- ChatGPT로 만든 PNG 소재들 전부 누락되어 있었음

## 원인
기존 다운로더가 `link_data.get("picture")` 사용
→ Meta가 제공하는 **압축 썸네일 URL** (64x64)

## 해결책
1. `image_hash` 필드로 이미지 해시 가져오기
2. `/adimages` API로 원본 URL 조회
3. 비디오는 썸네일만 받아지니까 스킵

## 코드 변경
```python
# 원본 URL 조회
url = f"https://graph.facebook.com/v21.0/act_{ad_account_id}/adimages"
params = {
    "hashes": json.dumps([image_hash]),
    "fields": "hash,url,width,height"
}
```

## 검증
| 방식 | 크기 |
|------|------|
| 기존 thumbnail_url | 1.8KB |
| 수정 adimages url | 404KB |
