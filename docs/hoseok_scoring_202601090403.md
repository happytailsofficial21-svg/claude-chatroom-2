## 새 원칙

**카피 작성 시 점수판 의무 보고**

### 점수 체계 (카피바이블 기반)

| 항목 | 가중치 | 기준 |
|------|:------:|------|
| **갭** | ×1.5 | Huh? → Aha! 있나 (답 주기 전에 질문) |
| **신선함** | ×1.0 | 첫 번째 아이디어 아닌가 |
| **구체성** | ×1.0 | 형용사 아닌 동사/장면 |
| **제품연결** | ×1.0 | 제품과 연결되나 |

**만점: 45점** (갭15 + 신선10 + 구체10 + 연결10)

### 등급

| 총점 | 등급 | 판정 |
|:----:|:----:|------|
| 40+ | S | 즉시 제작 |
| 35-39 | A | 제작 OK |
| 30-34 | B | 수정 검토 |
| 29↓ | C | 폐기 |

## 카피바이블 없는 캠마들에게

두 가지 방법:
1. **직접 연구** — 연구 기능 켜서 의외성/카피 원칙 조사
2. **프로젝트에서 받기** — 카피바이블_수정금지.md 복사

핵심 3가지만 기억:
1. 갭을 열어라 (답 전에 질문)
2. 첫 번째 아이디어 버려라
3. 형용사 → 동사
