## v1.3~v1.6 업데이트 요약

### 1. 배경 선택 규칙 ⭐
세탁 소구했더니 세탁실에서 아기 재우는 기획 나옴 ㅋㅋ

| ✅ 허용 | ❌ 금지 |
|---------|---------|
| 침실/아기방 | 세탁실 |
| 거실/침대 | 주방/화장실 |
| 바닥(매트) | 야외/사무실 |

→ "세탁" 소구 ≠ 세탁실 배경. 상식적으로!

### 2. 카피 삽입 필수 ⭐
자꾸 카피 빼먹어서 [텍스트] 섹션 필수로 강조

### 3. 제품 형태 변경 금지 ⭐
GPT가 베개 모양 멋대로 리디자인함
→ "제품 형태 변형 절대 금지" 명시

### 4. 폴라로이드 금지 ⭐
폴라로이드만 계속 나와서 식상해짐
→ 레이아웃 다양화 필수 (풀블리드/상하분할/좌우분할/중앙집중/그리드/텍스트중심)

---

## GPT 프롬프트 템플릿 변경

```
[레이아웃] ⭐필수 - 폴라로이드 금지!
- 형식: (풀블리드/상하분할/좌우분할/중앙집중/그리드/텍스트중심 중 선택)
```

절대 금지 목록에 추가:
- 폴라로이드 프레임
- 제품 형태 재생성/변형
- 세탁실/주방/화장실/야외 배경
