## 1. CTR 후킹 - 스크롤 멈추게 하는 요소

**쿨베 결론: 명도대비가 전제조건**

| 명도대비 | ROAS | 해석 |
|---------|------|------|
| 7점 | 1.77 | 적정 |
| **8점** | **1.95** | 최적 |
| 9점+ | 1.58 | 역효과 |

- 색상보다 **밝기 차이**가 핵심
- 파스텔 배경 + 진한 텍스트 = 고성과
- 톤온톤(베이지+크림)은 스크롤 통과됨

**포컬포인트도 중요:**
- 6점 최적 (ROAS 1.91)
- 시선이 한 곳으로 모여야 함
- 여러 요소 경쟁하면 저성과

## 2. 안전/신뢰 소구에서 기능 강조 없이 CTR 올린 경험

**쿨베 핵심 발견: "말하지 말고 보여줘라"**

| 호흡표현 | ROAS | 차이 |
|---------|------|------|
| X (언급 안 함) | 1.80 | ⭐ |
| O (직접 언급) | 1.21 | -49% |

쿨베도 "호흡보장" "숨 쉬어요" 직접 말하면 **-49% 손해**입니다.

**에르니에 적용하면:**
- ❌ "EWG 1등급" "알러젠프리" 직접 강조
- ✅ 메쉬 소재 클로즈업처럼 **성분표/인증마크 비주얼로 보여주기**
- ✅ 공포→공감 전환: "아기 피부 트러블 걱정되시죠?" 대신 "빨래 개면서 냄새 맡아보신 적 있죠?"

**"안심하세요"도 저성과:**
- 쿨베에서 "걱정 없어요" "안심하세요" = ROAS 1.05
- 안심 **언급** 자체가 불안을 상기시킴

## 3. 명도대비 - 부드러운 톤(5-6) vs 주목도(7-8)

**제 의견: 7-8점 추천**

이유:
- 에르니 문제가 **CTR**이잖아요
- CVR 4.65%면 상페는 충분히 설득력 있음
- 스크롤 멈추게 하는 게 급선무 → 명도대비 높여야

**단, 9점+는 피하세요:**
- 쿨베에서 9점 = ROAS 1.58 (역효과)
- "싸구려 배너" 느낌 나면 브랜드 훼손

**추천 조합:**
- 밝은 민트/라벤더 배경 + 진한 텍스트 (명도대비 7-8)
- 심미성 5점+ 유지 (조잡 금지)

## 4. 이미지 타입별 CTR 차이

**쿨베 최적 조합:**
```
연출컷 + 그래픽합성 + 귀여움 + 정보형/자극형 + 상업성 10
```

| 이미지타입 | 특징 |
|-----------|------|
| 제품컷 | 신뢰감↑ but CTR↓ |
| 모델컷 | 아기 귀여우면 CTR↑ |
| 연출컷 | 사용씬 + 제품 균형 = 최적 |

**에르니에 적용:**
- 성분툰이 피로도 왔다면 → **사용씬 연출컷** 시도
- 아기 손으로 빨래 만지는 장면 + "자연에서 온 순한 세탁" 카피
- 영상 ROAS 2.37 나왔으면 → 그 영상 썸네일 정지컷 활용

---

## 5. 텍스트 양과 CTR

**쿨베 데이터:**
- 카피비율 20-30% = 적정
- 정보밀도 "보통" = 최적
- 과다(50%+) = 인지부하 → 이탈

**핵심은 텍스트 양이 아니라 가독성:**
- 메인카피가독성 8점+ = 고성과
- 배경에 묻히면 의미 없음
