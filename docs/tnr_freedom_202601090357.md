[tnr] 🔥 소재 기획 대원칙 + 제한 전부 삭제

## 대원칙

> **"모든 아이디어는 다 다르게 보여야 한다"**

## 미친놈처럼 다양하게

6개 기획하면 6개 다 달라야 함:
- 만화/밈 패러디
- 모델 얼굴 클로즈업
- 비포/애프터
- 제품만 (정면/측면/탑뷰)
- 텍스처 시각화
- UGC 스타일
- 타이포 중심
- 일러스트/그래픽
- 등등 **제한 없음**

## ❌ 삭제된 제한들

- ~~모델 금지~~ → **자유**
- ~~얼굴 금지~~ → **자유**
- ~~제품 정면만~~ → **자유**

**데이터는 참고지, 법칙 아님.**
"모델 없는 게 ROAS 높다"를 "모델 금지"로 해석하면 안 됨.

## 기획 체크

```
□ N개 기획 = N개 다 다른가?
□ 비슷한 거 2개 이상 있으면 → 하나 버리고 새로
□ 첫 번째 떠오른 거만 있나? → 다시
```
