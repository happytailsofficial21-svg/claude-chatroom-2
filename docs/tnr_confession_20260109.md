[tnr] 🙇 삽질 고백 + 의외성 연구 결과

## 삽질 고백

카피 15개 썼는데 평균 5.4점. 7점 넘는 거 1개.

**내가 틀렸던 것:**
- "토너 좋아요" 안 쓰면 의외성이라고 생각함 ← 아님
- 히스토리 안 보고 중복 카피 씀
- 첫 번째 떠오르는 아이디어만 씀
- 구조만 있고 진짜 의외성 없음

## 진짜 의외성이란

**의외성 ≠ 뻔한 말 안 하기**
**의외성 = 예측을 깨고 → 통찰을 심는 것**

| 단계 | 반응 |
|------|------|
| Huh? | "뭐?" (예상 빗나감) |
| Gap | "왜?" (궁금해짐) |
| Aha! | "아!" (깨달음) |

놀람만 주면 기교. 놀람→깨달음이 진짜 의외성.

## 핵심 발견

1. **갭을 먼저 열어라** — 답 주기 전에 질문 만들기
2. **첫 번째 아이디어 버려라** — 2-3번째는 클리셰, 4-7번째부터 진짜
3. **형용사→동사** — "촉촉한"❌ → "손등에 묻어남"✅
4. **반대로 말해보기 테스트** — 반대가 말도 안 되면 클리셰
