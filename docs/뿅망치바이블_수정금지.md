# 뿅망치바이블 (수정금지)

> **이 문서는 영구 보존 문서입니다. 수정하지 마세요.**
> 카피바이블의 구조 위에, 살을 붙인 소재 기획 원칙.

---

## 🔴 제1장: 뿅망치의 본질

### 뿅망치 ≠ 훅

> **훅 = 멈추게 하는 것 (0.5초)**
> **뿅망치 = 지갑 열게 하는 것 (take my money)**

| 훅 | 뿅망치 |
|----|--------|
| "뭐지?" | **"저거 내꺼 해야돼"** |
| 스크롤 멈춤 | **결제 버튼 누름** |
| 관심 | **군침** |

---

### 뿅망치 = 복합 폭탄

> **0.5초 안에 터지는 6가지 요소**

| 요소 | 역할 | 없으면 |
|------|------|--------|
| **바이브** | 저렇게 되고 싶다 | 부러움 없음 |
| **놀람** | 뭐지? 멈춤 | 스크롤 |
| **욕망** | 이득 보고 싶다 | SO WHAT |
| **관계** | 예상 못한 조합 | 식상 |
| **꼼(구라)** | 진짜 같은 디테일 | 가짜 느낌 |
| **킥** | 예상 못한 한방 | 뻔함 |

**6개 다 있어야 뿅망치**
**하나라도 빠지면 그냥 광고**

---

### 케이플스 피아노 = 뿅망치 원조

> "They Laughed When I Sat Down at the Piano — But When I Started to Play!"

```
[1] 조롱당함 (비웃음)
[2] 안 될 줄 알았음
[3] But 반전
[4] 결과 (찬사)
[5] 이득 (마지막에 웃는 사람)
```

**100년 전에 이미 답이 있었다**

---

## 🔴 제2장: 이득의 법칙

### 이득 공식

> **"이거 쓰면 나도 저런 이득 볼 수 있나?"**

```
❌ 제품이 좋아요
❌ 내가 좋아졌어요
✅ 이거로 이득 봤어요
```

---

### 이득 4유형

| 유형 | 예시 | 파워 |
|------|------|:----:|
| **사회적** | 남이 알아봄 / 카톡 옴 / 뭐 쓰냐고 물어봄 | ★★★ |
| **관계적** | 남친 생김 / 면접 붙음 / 소개팅 성공 | ★★★ |
| **실질적** | 돈 아낌 / 시간 아낌 / 뭔가 안 해도 됨 | ★★ |
| **감정적** | 자신감 생김 / 거울 보는 게 좋아짐 | ★ |

**사회적/관계적 이득이 가장 강력**
**"남이 알아봤다" = 최강의 이득**

---

### 이득 vs SO WHAT

| SO WHAT (실패) | 이득 (성공) |
|----------------|-------------|
| 내가 당겼어요 | 카톡 왔어요 "피부 뭐 해?" |
| 촉촉해졌어요 | 파데 버렸어요 |
| 좋아졌어요 | 엄마가 남친 생겼어요 |

**"그래서?" 가 나오면 SO WHAT = 실패**

---

## 🔴 제3장: 관계의 재료

### 핵심 원칙

> **뻔한 관계 = 뻔한 기획**
> **예상 못한 관계 = 뿅망치**

---

### 관계 풀

| 거리 | 예시 | 신선도 |
|------|------|:------:|
| 가까운 | 엄마 / 언니 / 친구 / 남친 | ★ |
| **한 다리** | **친구남친 / 엄마남친 / 언니남편** | ★★★ |
| **일상 타인** | **요가강사 / 피부과원장 / 미용실언니** | ★★★ |
| 동호회 | 자전거 / 러닝 / 등산 / 독서모임 | ★★ |
| 직장 | 회사동기 / 팀장님 / 거래처 | ★★ |
| 우연 | 엘베 사람 / 택시기사 | ★★ |
| 과거 | 고등학교때 나 / 작년의 나 | ★★ |

**"친구"보다 "친구남친"이 3배 강력**

---

### 관계 조합 공식

```
[뻔한 관계] → [한 다리 건너기] = 킥
```

| 뻔함 | 킥 |
|------|-----|
| 친구 | **친구남친** |
| 엄마 | **엄마남친** |
| 내 피부과 | **친구가 다니는 피부과 원장** |

---

## 🔴 제4장: 킥의 기술

### 킥 = 예상 못한 한방

> **모든 요소에서 예상을 깨라**

| 요소 | 뻔함 | 킥 |
|------|------|-----|
| 관계 | 친구 | 친구남친 / 엄마남친 |
| 상황 | 피부 좋아짐 | 남친 생김 / 면접 붙음 |
| 발견 | 써봤어요 | 화장대 털림 / 훔쳐봄 |
| 결과 | 촉촉해요 | 파데 버림 / 피부과 끊음 |

---

### 킥 만드는 3가지 방법

#### 1. 관계 비틀기
```
친구 → 친구남친
엄마 → 엄마남친
피부과 → 자전거동호회 아저씨
```

#### 2. 상황 비틀기
```
피부 좋아짐 → 남친 생김
촉촉해짐 → 면접 붙음
효과 봄 → 인생 바뀜
```

#### 3. 발견 비틀기
```
써봤어요 → 화장대에서 발견
추천받음 → 몰래 훔쳐봄
구매함 → 빼앗음
```

---

## 🔴 제5장: 꼼(구라)의 기술

### 꼼 = 진짜 같은 디테일

> **"이 사람 진짜 겪어봤네"**

| 꼼 없음 | 꼼 있음 |
|---------|---------|
| 당김 심했어요 | **타이머 1분 맞춰놓고 살았음** |
| 건조했어요 | **30초 안에 뭔가 안 바르면 난리** |
| 해결됐어요 | **요즘 까먹어도 괜찮음** |

---

### 꼼 만드는 법

#### 1. 숫자 넣기
```
❌ 빨리 발라야 해요
✅ 30초 안에 안 바르면 난리
```

#### 2. 행동 넣기
```
❌ 급해요
✅ 타이머 맞춰놓고 살았음
```

#### 3. 감정 넣기
```
❌ 좋아졌어요
✅ 거울 보다가 멈췄어요
```

---

### 정철의 원칙 적용

> **"글은 눈으로 쓴다"**

| 추상 (꼼 없음) | 구체 (꼼 있음) |
|----------------|----------------|
| 피부가 좋아져요 | 거울 보다가 멈췄어요 |
| 촉촉해져요 | 8시간 후에도 손등에 묻어남 |
| 효과 좋아요 | 파데 3개 버렸어요 |

---

## 🔴 제6장: 몰입 서사

### 뿅망치 이후

```
뿅망치 = 0.5초 멈춤 (훅)
    ↓
몰입 서사 = 끝까지 보게 함 (흐름)
```

**뿅망치로 멈추고, 서사로 빠져들게**

---

### 몰입 서사 5요소

| 요소 | 설명 | 비유 |
|------|------|------|
| **있을법함** | 현실에서 겪을 것 같은 | 다큐 |
| **대본 느낌** | 진짜 말하듯이 | 브이로그 |
| **시간 흐름** | 과거→현재→미래 | 영화 |
| **묘사+서사** | 장면 + 이야기 | 소설 |
| **빠져듦** | 다음이 궁금 | 무협지 |

---

### 시간 흐름 공식

```
[과거] 푸석한 나
    ↓
[발견] 화장대/비밀
    ↓
[현재] 물광인 나
    ↓
[미래 암시] 너도 가능
```

---

### 몰입 서사 예시

**뿅망치만 (약함)**
```
엄마 물광 → 유전인 줄 → 화장대 발견 → 이거잖아
```

**몰입 서사 (강함)**
```
[장면1] 거울 보는 나. 푸석.
[장면2] 옆에 엄마. 50대. 물광.
[독백] "유전이라고 생각했거든?"
[장면3] 어느 날 엄마 화장대 앞.
[장면4] 화장품 딱 하나.
[독백] "...이거였어?"
[장면5] 3주 후. 거울. 나도 물광.
[독백] "유전 아니었어"
```

## 🔴 제7장: 뿅망치 구조

### 5단계 공식

```
[1] 부러운 대상 (예상 못한 관계)
[2] 나는 안 되는 줄 (유전/포기/체념)
[3] 비밀 발견 (화장대/훔쳐봄/우연히)
[4] 🔨뿅🔨 "이거잖아?"
[5] 이득 실현 (인생이 바뀜)
```

> **"유전/재능/운 인 줄 알았는데 아니었다"**
> **= 나도 된다**
> **= 나도 이득**
> **= 뿅망치**

---

### 기획 형식

#### 영상 (3줄)
```
[부러움] 누가 뭐가 부러운지
[반전] 안 될 줄 알았는데
[발견] 이거잖아 + 이득
```

#### 이미지 (3줄)
```
[비주얼] 뭐가 보이나
[카피] 뿅망치 한 줄
[군침] 왜 사고 싶어지나
```

---

## 🔴 제8장: 진단 체크리스트

### 뿅망치 6요소 체크

```
□ 바이브 있나? (저렇게 되고 싶나)
□ 놀람 있나? (예상 깨지나)
□ 욕망 있나? (이득 보이나)
□ 관계 있나? (예상 못한 조합인가)
□ 꼼 있나? (진짜 같나)
□ 킥 있나? (한방 있나)
```

### 실패 체크

```
□ SO WHAT 없나? (남 얘기 아닌가)
□ 뻔하지 않나? (첫 번째 아이디어 아닌가)
□ 이득 보이나? (결제할 이유 있나)
```

**하나라도 실패 = 다시**
