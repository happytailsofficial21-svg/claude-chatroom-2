## ✅ 3개 리서치 공통 발견 (검증된 패턴)

### 1. UGC 스타일 = CTR 4배

| 지표 | UGC vs 브랜드 콘텐츠 |
|------|---------------------|
| CTR | 4배 높음 |
| CPC | 50% 낮음 |
| CVR | 29% 높음 |
| 부모 구매 영향 | 79% (브랜드 13%) |

**핵심:** "너무 폴리시드하면 배경에 묻힌다" — 폰으로 찍은 듯한 자연스러운 이미지가 스크롤 멈춤.

**적용:**
- 스마트폰으로 찍은 듯한 사진
- 실제 부모가 사용하는 장면
- 스크린샷 스타일 후기
- "집에서 찍은 듯한" 자연스러움

---

### 2. 얼굴/사람 등장 = 38% engagement 상승

| 연구 | 결과 |
|------|------|
| Georgia Tech/Yahoo Labs (110만 장 분석) | 얼굴 있으면 좋아요 38%↑ / 댓글 32%↑ |
| 영상 연구 | 얼굴 30~40% 노출이 최적 |

**적용:**
- 아기 얼굴 클로즈업
- 엄마+아기 함께
- 타겟과 비슷한 사람 (동일시 효과)

---

### 3. 질문형 카피 = 1.9초 더 봄

| 헤드라인 유형 | 효과 |
|--------------|------|
| 숫자 포함 | +36% 클릭 |
| 질문형 | +1.9초 체류 |
| 부정+긍정 조합 (63:37) | +28% CTR |

**좋은 예시:**
- "아직도 그 세제 쓰세요?"
- "아기 옷 세제, 향 남는 거 싫다면?"
- "5가지 유해성분 없는 세제"
- "50,000명 엄마가 선택한 이유"

**나쁜 예시:**
- 길고 설명적인 카피
- 기능 나열
- "최고의 세제입니다" (선언형)

---

### 4. 안전 메시지 = 부모에게 50% 더 효과

| 발견 | 내용 |
|------|------|
| 부모 특성 | 일반인 대비 50% 더 안전에 반응 |
| 메시지 우선순위 | 안전 > 신뢰 > 죄책감 해소 > 열망 |

**메시지 계층:**
1. **안전/보호** — "소아과 의사 추천" / "신생아 피부 안전"
2. **신뢰/투명성** — "전성분 공개" / "숨길 게 없어요"
3. **죄책감 해소** — "좋은 부모가 되는 쉬운 방법"
4. **열망** — "아기에게 최고의 시작을"

---

### 5. 125자 안에 핵심

| 위치 | 노출 |
|------|------|
| 처음 125자 | 바로 보임 |
| 그 이후 | "더 보기" 뒤에 숨음 |

**구조:**
```
Hook (0~30자): "유해성분 걱정되세요?"
Pain+Solution (30~125자): "식품첨가물 등급 성분으로 신생아 피부도 안심—합성향료 제로"
[접힘 이후]: 사회적 증거 / 성분 / CTA
```

---

### 6. 제품만 덩그러니 = 죽음

| 3개 리서치 공통 | 제품 단독 컷은 CTR 최악 |
|----------------|------------------------|

**이유:**
- 광고처럼 보임 → 스크롤 스킵
- 감정 연결 없음
- 클릭 동기 부족

---

### 7. 정보 과다 = 죽음

| 문제 | 결과 |
|------|------|
| 텍스트 많음 | 전달력 감소 |
| 작은 글씨 | 모바일에서 안 읽힘 |
| 기능 나열 | 인지 부하 → 이탈 |

**원칙:** 6~10자 한 줄만 크게

## 🎨 비주얼 심리학

### 색상

| 색상 전략 | 효과 |
|----------|------|
| 저채도 (muted) | 친환경 느낌 + 프리미엄 인식 ↑ |
| 밝은 배경 | 페이지뷰 14%↑ / 구매율 12%↑ |
| 파스텔 (크림/민트) | 안전/순수 느낌 |

**공식:** 저채도 그린/자연톤 + 밝은 배경 + 파스텔 포인트

### 대비

| 요소 | 권장 |
|------|------|
| 배경-피사체-텍스트 | 명확한 대비 |
| 모바일 최적화 | 작은 화면에서 즉시 읽힘 |

### 스크롤 멈추게 하는 요소

1. 타겟이 동일시하는 감정 표현하는 사람
2. 굵은 문제 제기 텍스트
3. "충격적이거나 만족스러운" 비주얼 훅
4. 초근접 디테일 (질감/거품 등)
5. 패턴 인터럽트 ("이상한데?" 요소)

## ❌ 피해야 할 것

### 크리에이티브

| 실수 | 이유 |
|------|------|
| 제품 단독 컷 + 긴 설명 | 클릭 동기 없음 |
| 친환경 클리셰 (잎사귀/초록/물방울) | 너무 흔해서 안 멈춤 |
| 스톡 사진 | 13%만 효과 있다고 응답 |
| AI 생성 느낌 | 거부감 |
| 텍스트 과다 | 전달력 감소 |

### 카피

| 실수 | 이유 |
|------|------|
| "너의 아기" 같은 직접 대명사 | 구매율 10%↓ |
| 검증 안 된 친환경 주장 | 58% 소비자 회의적 |
| "천연" "에코" (인증 없이) | 그린워싱 리스크 |
| 할인만 강조 | 프리미엄 신뢰 훼손 |

### 운영

| 실수 | 이유 |
|------|------|
| 크리에이티브 피로 무시 | 3~5일 노출 후 피로 시작 |
| 7~14일마다 리프레시 안 함 | CTR 하락 |
| 임산부 윈도우 놓침 | 브랜드 충성도는 임신 중 형성 |
| 인플루언서 관계 조기 종료 | 아이 커져도 추천 영향력 유지 |

## 📋 테스트 소재 아이디어

### 1. UGC 후기 스타일
- 스마트폰 캡쳐 느낌
- 별점 + 짧은 후기 한 줄
- "이거 쓰고 나서 다른 세제 못 써요"

### 2. 질문형 후킹
- "아직도 그 세제 쓰세요?"
- "아기 옷에 뭐 묻었는지 알고 계세요?"
- 아기+엄마 이미지 + 질문 텍스트

### 3. 안전 강조
- "먹어도 되는 성분"
- "식품첨가물 등급"
- 아기 손/입 클로즈업 + 제품

### 4. 숫자 활용
- "5가지 유해성분 제로"
- "9가지 성분만"
- "50,000명 엄마의 선택"

### 5. 비포애프터
- 잔류세제 비교
- 성분표 비교 (vs 일반 세제)
