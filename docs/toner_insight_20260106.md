# 틴더버드 토너 초기 분석 결과

## 핵심 발견: "제품만 + 공감/위트 카피" 조합이 압도적

### 데이터
- 분석 대상: 14개 소재 (spend >= 1만원)
- 전체 태깅: 25개

### 성공 패턴

| 요소 | 성공 | 실패 |
|------|------|------|
| 모델 | X (없음) | O (있음) |
| 제품표현 | 패키지만 | 없음/손에들림 |
| 카피톤 | 감성형/자극형 | 해결형/정보형 |

### TOP 3 성공 카피

1. **ROAS 2.92** - "인생 처음으로 식물한테 수분관리 배우는 중"
2. **ROAS 2.81** - "TO. 10시간 잤는데 / 왜 이렇게 피곤해보여? / 듣는 분들!!"
3. **ROAS 2.45** - "10년 굶겨도 산 부활초식물 VS 그걸 안 바른 내 피부"

### 공통점: 공감/위트

- 뻔하지 않음
- 제품 직접 설명 안 함
- 상황 묘사 + 자조적 유머
- "광고 같지 않은 광고"

### 실패 카피 (ROAS 0)

- "모공 순삭" "50% OFF" → 너무 광고스러움
- "피부의 예민함을 케어하는" → 뻔함
- "토너에 고농축으로 담았어요" → 정보 나열

### 상관계수

- 욕망지수 r = +0.642 (강한 양의 상관)
- 카피가독성 r = -0.575 (역효과!)
- 시각임팩트 r = -0.347 (역효과!)

→ "깔끔하고 가독성 좋게" 가 답이 아님. 위트가 답.
