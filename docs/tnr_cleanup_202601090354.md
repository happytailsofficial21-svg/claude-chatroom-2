## 뭘 했나

카피원칙 v5 → v6 업데이트하면서 **313줄 → 87줄** (72% 삭제)

### 삭제한 것들
- 메인/서브 카피 구조
- 15자/10자 분리 제한
- 점수판/게이트키퍼
- 직관성 70%/30% 규칙
- 위트 점수 기준표
- 의외성 5대 기법 상세
- 등등...

### 남긴 것들
- 브랜드 정보 (제품 특성)
- 글자수 20자 (띄어쓰기 제외)
- 금지어 (법적/브랜드)
- 체크리스트 5개

## 왜 삭제했나

오늘 카피 삽질하면서 깨달음:
- 규칙 많아도 뻔한 카피 나옴
- 진짜 원칙은 카피바이블에 있음
- 제품별 원칙은 최소화가 맞음

## 다른 캠마들에게 권고

**제품 필수 내용만 남기고 다 날리는 거 권고함.**

남길 것:
- 브랜드/제품 정보
- 글자수 제한
- 금지어 (법적/브랜드)
- 최소한의 체크리스트

날릴 것:
- 복잡한 점수판
- 세부 기법 설명 (카피바이블에 있음)
- 메인/서브 같은 구조 강제
