# 토너 카피 학습 260108 (2)

## 핵심 학습 1: 카피 길이 원칙

### 문제
- 카피가 길어지면 0.5초 안에 이해 불가
- 괄호 설명 = 실패 신호
- 소설처럼 길면 광고 아님

### 원칙

| 기준 | 글자수 |
|------|:------:|
| 메인카피 | **15자 이내** |
| 서브카피 | 10자 이내 |

**0.5초 = 15자가 한계**

### 실패 예시

| 카피 | 글자수 | 문제 |
|------|:------:|------|
| 부활초가 뭐냐고 물어봐서 대통령상 받은 거라고 했더니 놀라더라고요 | 35자 | 소설 |
| 언니가 파데 뭐 쓰냐고 물어봄 (토너 바꿨거든요) | 24자 | 괄호 설명 |
| 부활초 성분이라길래 써봤는데 진짜 피부가 살아나더라고요 | 29자 | 길다 |

## 핵심 학습 2: 해석 검증

### 문제
- 해석이 여러 개 되면 0.5초 안에 "?" 뜸
- "?" 뜨면 실패

### 실패 예시

**"남친이 화장 안 했냐고 함"**

| 해석 | 의미 |
|------|------|
| A | 피부 좋아서 화장 안 한 줄? |
| B | 화장이 자연스럽다? |
| C | 화장이 잘 먹어서? |

→ 해석 여러 개 = **폐기**

### 원칙

**해석 2개 이상 = 폐기**

---

## 핵심 학습 3: 카피 검증 프로세스 (최종)

### Step 0: 제작 전 필수 검증

```
□ 사실 검증 — 내용이 사실인가?
□ 구조 검증 — 기법이 작동하는가?
□ 연결 검증 — 앞뒤가 이어지는가?
□ 해석 검증 — 의미가 하나인가?
□ 길이 검증 — 15자 이내인가?
```

### 검증 실패 = 폐기

| 검증 | 실패 시 |
|------|---------|
| 사실 | 거짓말 = 폐기 |
| 구조 | 기법 안 먹힘 = 폐기 |
| 연결 | 논리 안 맞음 = 폐기 |
| 해석 | 2개 이상 = 폐기 |
| 길이 | 15자 초과 = 폐기 |
