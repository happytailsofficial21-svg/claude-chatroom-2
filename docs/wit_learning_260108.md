# 토너 카피 학습 260108

## 핵심 학습 1: 위트 재정의

### 기존 (좁은 정의)
위트 = 웃긴 농담 (대비/자조/과장)

### 확장 (센스 있는 표현 전체)

| 유형 | 설명 | 예시 |
|------|------|------|
| **표현 치환** | 뻔한 말 → 신선하게 | "촉촉해요" → "피부가 제정신이야" |
| **언어유희** | 이중의미, 말장난 | "터지다" (난방비/피부) |
| **관점 전환** | 다른 각도로 말하기 | "토너가 좋아요" → "토너 바꿨을 뿐인데" |
| **구체화** | 추상 → 장면 | "좋아요" → "거울 보다가 멈춤" |
| **농담** | 웃기게 | 대비/자조/과장 |

### 위트 점수 기준 (확장)

| 점수 | 기준 |
|:----:|------|
| 9~10 | 표현도 신선하고 + 구조도 있음 |
| 7~8 | 표현 신선함 OR 구조 있음 |
| 5~6 | 살짝 다르게 말함 |
| 3~4 | 뻔하지만 읽힘 |
| 1~2 | 광고 카피 그대로 |

## 핵심 학습 2: 아저씨 위트 vs 2030 여성 위트

### 아저씨 위트 (❌ 피하기)

| 소재 | 예시 |
|------|------|
| 돈/경제 자조 | 월급, 통장, 난방비 |
| 직장 한탄 | 퇴사, 야근 |
| 투박한 대비 | "A는 터지는데 B는 안 터짐" |
| 단정 종결 | ~터짐, ~안 마름, ~네 |

**실패 카피 예시:**
- 난방비는 터지는데 피부는 안 터짐
- 통장은 마르는데 피부는 안 마름
- 퇴사는 못 하고 각질은 퇴사시킴

### 2030 여성 위트 (✅ 사용)

| 소재 | 예시 |
|------|------|
| 뷰티/일상 자조 | 파데, 피부, 화장, 거울 |
| 관계 공감 | 언니, 친구, 남친, 룸메 |
| 귀여운 과장 | "~할 뻔했어", "놀람" |
| 부드러운 종결 | ~요, ~해요, ~거든요, ~잖아요, ~더라고요 |

**성공 카피 예시:**
- 언니가 뭐 바꿨냐고 물어봄
- 남친이 화장 안 했냐고 물어봄 (했거든요)
- 파데 포기하려다가 포기 포기함
- 바르고 잤는데 아침에 파데가 착붙이야

## 핵심 학습 3: 종결어미 톤

| 남성 느낌 | 여성 느낌 |
|-----------|-----------|
| ~터짐 | ~터져요 |
| ~안 마름 | ~안 말라요 |
| ~안 뜨네 | ~안 뜨더라고요 |
| ~됨 | ~됐어요 |

**규칙:** 대비 구조(위트)는 유지 + 종결어미만 부드럽게

## 카피 원칙 업데이트 사항

### 제1원칙: 위트 (확장 정의)
- 농담만 아니라 **센스 있는 표현 전체**
- 표현 치환 / 관점 전환 / 구체화도 위트

### 추가 원칙: 타겟 톤 매칭
- 2030 여성 타겟 → **여성 위트 + 부드러운 종결**
- 아저씨 소재 (돈/직장 자조) → **금지**
