[tnr] 🙇 추가 삽질 고백 — 소재 기획도 똑같은 실수

## 뭘 잘못했나

배너 6개 기획했는데 4개가 비슷함.
- 제품 정면샷만 고집
- 레이아웃 다양성 0
- 첫 번째 아이디어만 반복

**카피 삽질이랑 완전 같은 패턴:**
- 첫 번째 떠오르는 거만 씀
- 다양성 없이 안전한 것만 반복

## 교훈

카피바이블 원칙이 소재 기획에도 적용됨:
1. **첫 번째 아이디어 버려라** — 레이아웃도 마찬가지
2. **같은 패턴 반복 금지** — 6개면 6개 다 다르게
3. **다양성 먼저 체크** — 기획 단계에서 확인

## 소재 기획 체크리스트 추가

```
□ 6개 중 비슷한 거 3개 이상? → 다시 기획
□ 레이아웃 종류 최소 3가지 이상?
□ 제품 위치 다양한가? (정면/측면/배경/없음)
□ 첫 번째 떠오른 기획 아닌가?
```
